#dev-env-setup.sh
This is a an install script to install the following packages on a new Ubuntu 18.04 system.Tested on Linux Mint 19.1
- git, curl
- gcc, g++ and make
- gcc-arm-none-eabi
- gdb
- vim
- zsh
- qemu + virt-manager GUI
- Chrome
- VSCode
- Nodejs
- Angular
- MongoDB
- Docker
- Python3 and pip
- VSCode extensions
- PowerTOP
- TLP
