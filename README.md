A quick and dirty script to install my most used programs on a new Ubuntu system
This installs
- git, curl
- gcc, g++ and make
- gcc-arm-none-eabi
- gdb
- vim
- zsh
- qemu + virt-manager GUI
- Chrome
- VSCode
- Nodejs
- Angular
- MongoDB
- Docker
- Python3 and pip
- VSCode extensions
- PowerTOP
- TLP
